Introduction au PHP

PHP (officiellement le signe est un acronyme récursif pour PHP Hypertext processor) est un langage de scripts généralistes et open Source, spécialement conçu pour le développemnt d'application WEB.

1- Aquoi sert le PHP

Comme nous l'avons lu dans le texte d'introduction, le PHP sert à développer des applications Web.Grâce à ce langage, nous pouvons faire tout type de site car il n'y a plus de limite.En effet, le PHP est un langage fonctionnel à l'inverse du HTML et du CSS.

Nous pourrons maintenant grâce au PHP, créer tous les types de sites que nous souhaitons:moteur de recherche de type Google, réseaux sociaux comme Facebook, des plateformes multimedias comme Youtube, ou encore des sites d'informations comme wikipédia. Mais son utilisation ne se limite pas à ça car nous pourrons aussi développer des sites de e-commerce, des espaces membres, des pages d'administation etc.

Pour mieux comprendre le PHP va être l'élément qui permet à ce que l'on a créé grâce aux autres langages de fonctionner. Si on prend l'exemple d'un formulaire : on le déclare en HTML , on le style grâce au CSS ou à un framework de type Boostrap.Si nous voulons récupérer les données qui se trouvent dans le formulaire nous devrons utiliser le PHP qui va permettre de traîter les données.

2- Site dynamique

Le PHP nous sert en fait à crééer un site web dynamique . Un site web dynamique va être un site dont les infos proviennent s'une BDD et pour cela, on va apprendre à faire fonctionner le PHP et le SQL ensemble.

Attention, il ne faut pas confondre un site WEB DYNAMQUE avec un site de mouvement et d'animations en couleur cliquable résalisable grâce au JS.

Nous utiliserons à partir de maintement l'appellation de Site dynamique pour les sites sont les infos sont enregistrés dans une BDD et qui possède deux interfaces :

  1. Une interface pour le front ou on retrouvera l'affichage du site "normal", le côté pour les utilisateurs et les clients
  2. Une interface pour le back qui nous permettra d'assurer la gestion de notre base de données et les réglages du front => on l'appellera le back office ou la page d'administration .

Un site dynamique n'utilise pas qu'un seul langage. Il faut marier le HTML, le CSS et le SQL ( JS etc.)

4- Serveur et différences entre PHP procédural et orienté objet

Un serveur est généralement un ordinateur qui héberge un site mis à disposition par un hébergeur. Cet ordinateur reste toujours allumé pour que le site soit consultable 7j/7 et 24h/24.

Il existe deux technique de développement avec le PHP :

  • L'approche procédurale ou le code va être écrit de façon séquentielle.
  • L'approche orienté objet : Le code est alors encapsulé dans des méthodes de classe et fonctionne grâce à l'interaction d'objet.

Certaines méthodes de base doivent être forcément apprises grâce au procédural . Il n'est pas possible d'aller ' plus loin' grâce à l'orienté objet . Ce sont en fait deux méthodologies différentes mais qui permettent de faire la même chose.

Certaines grandes entreprises préfèrent travailler avec l'approche orientée objet car elle encourage le travail de groupe et facilite les mises à jour sur des travaux longue durée.

Ces deux méthodes peuvent être utilisées pour développer un site "FROM SCRATCH" ( de zéro), dans un CMS(comme Wordpress, wix) ou encore dans un Framework (Symfony utilise naturellement l'orienté objet cependant il est possible de faire du procédural dedans).
